About BugFire

This application is a non-commercial pet-project of Ivan Ivanov. Initially, it was developed for personal use but now it's an open-source project which is free to use

There were several main goals for this project. Main features are done and ready to use but more is coming.

  • Tracking bugs of any kind
  • Managing and expanding a team
  • Collecting feedback from the product users

When a project gets too big, it is difficult to keep control of everything. BugFire is developed with the real purpose of managing bugs and task to help developers accomplish their goals. This application is still in development, and will keep getting better with the main idea of simple but smart design.

Frequently Asked Questions

If you're going to use this app, you're probably curious about some sensitive details of this application, especially how the data is stored.

  • Will be any enetered data available publically or to someone out of this app?

    No. The data is stored in private MySQL database. Nobody have an access to it, and will never have.

  • How passwords are stored?

    After entering a password it will be hashed and salted several times. The original password will be deleted. The database stores only encrypted passwords.

  • What method is used to hash a password?

    The method is called "sha256", it uses the pbkdf2 standart. It is also gets salted several times, after hashing the password.
